# Cardinal Six Cyber Blog Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a trust-first Astro blog with a filterable two-lane index (`Practical` and `Research`), file-based Markdown/MDX publishing, and two sample posts that fit the existing Cardinal Six site.

**Architecture:** Blog content lives in an Astro content collection under `src/content/blog/`, validated by a typed schema. A dedicated blog index page renders the content list with lane/topic filtering, while a reusable `BlogPostLayout` renders both Markdown and MDX entries with lane-aware metadata, long-form reading styles, and soft CTAs.

**Tech Stack:** Astro 6, `astro:content`, `@astrojs/mdx`, Markdown/MDX, existing `BaseLayout`/`Nav`/`Footer`, existing global CSS tokens

**Spec:** `docs/superpowers/specs/2026-04-17-blog-design.md`

```ts
import { defineCollection, z } from 'astro:content';

const blog = defineCollection({
    type: 'content',
    schema: z.object({
        title: z.string(),
        description: z.string().max(180),
        publishDate: z.coerce.date(),
        updatedDate: z.coerce.date().optional(),
        author: z.string(),
        lane: z.enum(['practical', 'research']),
        topics: z.array(z.string()).min(1),
        featured: z.boolean().default(false),
        draft: z.boolean().default(false),
        heroStyle: z.string().optional(),
        heroImage: z.string().optional(),
        citations: z.boolean().default(false),
        diagrams: z.boolean().default(false),
        canonicalUrl: z.string().url().optional(),
    }),
});

export const collections = { blog };
```

- [ ] **Step 2: Add the client-side filter script**

Append this script block in `src/pages/blog/index.astro`:

```astro
    <script>
        (function() {
            var activeLane = '';
            var activeTopic = '';
            var filters = document.querySelectorAll('.blog-filter');
            var topics = document.querySelectorAll('.blog-topic');
            var cards = document.querySelectorAll('.blog-feed .blog-card');

            function applyFilters() {
                cards.forEach(function(card) {
                    var laneMatch = !activeLane || card.getAttribute('data-lane') === activeLane;
                    var topicData = card.getAttribute('data-topics') || '';
                    var topicMatch = !activeTopic || topicData.split('|').includes(activeTopic);
                    card.style.display = laneMatch && topicMatch ? '' : 'none';
                });
            }

            filters.forEach(function(button) {
                button.addEventListener('click', function() {
                    activeLane = button.getAttribute('data-lane') || '';
                    filters.forEach(function(item) { item.classList.remove('active'); });
                    button.classList.add('active');
                    applyFilters();
                });
            });

            topics.forEach(function(button) {
                button.addEventListener('click', function() {
                    activeTopic = button.getAttribute('data-topic') || '';
                    topics.forEach(function(item) { item.classList.remove('active'); });
                    button.classList.add('active');
                    applyFilters();
                });
            });
        })();
    </script>
</BaseLayout>
```
